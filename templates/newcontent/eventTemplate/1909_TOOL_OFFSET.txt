
h1. @TOOL_OFFSET@

h2. Part 2 Content

* Category: @EVENT@

|_.Type/Sub Type|_.Description|Units|
|@TOOL_OFFSET@&nbsp;&nbsp;&nbsp; |A reference to the tool offset values applied to the active cutting tool associated with a piece of equipment.@
@
The valid data value *MUST* be a text string.@
@
The reported value returned for @TOOL_OFFSET@ identifies the location in a table or list where the actual tool offset values are stored.@
@
A subType *MUST* always be specified.|MILLIMETER|
|>.@RADIAL@|A reference to a radial type tool offset value.|MILLIMETER|
|>.@LENGTH@|A reference to a length type tool offset value.|MILLIMETER|
