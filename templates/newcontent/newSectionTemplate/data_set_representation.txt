h1. @DATA_SET@ _(Section Name)_

* Part : Part 2

h2. Parent : @representation Attribute for DataItem@

Add a new representation for associative arrays of values for data items. Any data item can have a representation of @DATA_SET@ that will append the suffix @DataSet@ to @type@ as described in the transform from _Part_ 2 enumerations to _Part_ 3. 

The @DataSet@ data items will be represented as list of @Entry@ elements that have the attributes shown in {ref:attributes-for-dataset}.

{table:attributes-for-dataset}

The value of the @Entry@ element will have the same facet or controlled vocabulary from the @type@'s default representation.
